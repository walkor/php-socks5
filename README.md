# socks5-proxy
Socks5 proxy written in PHP based on [workerman](https://github.com/walkor/Workerman).

## Install
1. ```git clone https://github.com/walkor/php-socks5```

2. ```composer install```

## Start
php start.php start -d

## Stop
php start.php stop

## Status
php start.php status
